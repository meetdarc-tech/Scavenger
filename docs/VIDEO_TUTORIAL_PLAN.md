# Video Tutorial Series Plan

This document outlines the planned video tutorial series for the Scavngr platform.

## Series Overview

The Scavngr video tutorial series teaches users how to use the platform as each of the three participant roles (Recycler, Collector, Manufacturer) and developers how to build on top of the smart contracts.

**Hosting**: Videos are hosted on YouTube (Scavngr channel) and embedded in the documentation website at https://docs.scavngr.io.

## Video Hosting

- **Platform**: YouTube (unlisted during review, public on release)
- **Playlist structure**: One playlist per series
- **Embedding**: Tutorials are embedded in the documentation website using Docusaurus MDX
- **Analytics**: YouTube Analytics + plausible.io for docs page engagement

---

## Accessibility

- Closed captions (auto-generated + manually corrected) on all videos
- Full transcript provided alongside each video in the docs
- Videos include audio descriptions of on-screen actions
